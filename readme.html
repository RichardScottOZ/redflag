üí° redflag aims to be an automatic safety net for machine learning datasets. The vision i sto accept input of a Pandas DataFrame or NumPy ndarray (one for each of the input X and target y in a machine learning task). redflag will provide an analysis of each feature, and of the target, including aspects such as class imbalance, outliers, anomalous data patterns, threats to the IID assumption, and so on. InstallationÔÉÅ

You can install this package with pip:

pip install redflag

ExperimentalÔÉÅ

Installing scikit-learn allows you to access some extra options for outlier detection.

pip install redflag[sklearn]

ExampleÔÉÅ

redflag is currently just a collection of functions. Most of the useful ones take a single column of data (e.g. a 1D NumPy array) and run a single test. For example, we can do some outlier detection:

>>> import redflag as rf
>>> data = [-3, -2, -2, -1, 0, 0, 0, 1, 2, 2, 3]
>>> rf.has_outliers(data)
array([], dtype=int64)
>>> rf.has_outliers(3 * data + [100])
array([100])

See the notebook Using_redflag.ipynb for several other examples.

TestingÔÉÅ

You can run the tests (requires pytest and pytest-cov) with

python run_tests.py

Most of the tests are run with doctest.

BuildingÔÉÅ

This repo uses PEP 517-style packaging. Read more about this and about Python packaging in general.

Building the project requires build, so first:

pip install build

Then to build redflag locally:

python -m build

The builds both .tar.gz and .whl files, either of which you can install with pip.
