redflag.feature.zscore_outliers(a, sd=3, limit=4.89163847)Â¶

Find outliers using Z-scores.

Expect points outside:

  • 1 sd: expect 31.7 points in 100

  • 2 sd: 4.55 in 100

  • 3 sd: 2.70 in 1000

  • 4 sd: 6.3 in 100,000

  • 4.89163847 sd: 1 in 1 million

  • 5 sd: 5.7 in 10 million datapoints

  • 6 sd: 2.0 in 1 billion points

Parameters

  • a (array) â The data.

  • sd (float) â The number of standard deviations to use as the threshold for ordinary outliers.

  • limit (float) â The number of standard deviations to use as the threshold for âextremeâ outliers.

Returns

outliers and extreme outliers.

Return type

tuple

Examples

>>> data = [-3, -2, -2, -1, 0, 0, 0, 1, 2, 2, 3]
>>> ratio, _, _, _ = zscore_outliers(data)

  • âklâ: Kullback-Leibner divergence

It can also be a function returning a divergence.

Parameters

  • method (str or function) â The method to use.

Returns

Return type
